# Slide 16 – References

## Resources

- [keda.sh](https://keda.sh/) – official Kubernetes Event-Driven Autoscaling documentation and scaler catalog.
- [Azure Functions on Kubernetes with KEDA](https://learn.microsoft.com/azure/functions/functions-kubernetes-keda) – Microsoft guidance for running Functions with KEDA.
- [KEDA on Azure Kubernetes Service](https://learn.microsoft.com/azure/aks/keda-about) – managed cluster integration notes.
- [Scaling Workloads on Microsoft Azure with KEDA](https://opensource.microsoft.com/blog/2023/08/24/scaling-workloads-on-microsoft-azure-with-keda/) – engineering blog highlighting customer patterns.

Encourage attendees to bookmark these links and share them with teammates exploring event-driven architectures on Kubernetes.

## How to Keep Learning

- Subscribe to the [KEDA community calendar](https://keda.sh/community/) for release notes and community meetings.
- Explore sample repositories such as [Azure Functions on Kubernetes samples](https://github.com/Azure/azure-functions-on-kubernetes) to see production-aligned implementations.
- Review observability tooling (Prometheus, Azure Monitor, Grafana) so you can visualise scaling decisions in real time.
